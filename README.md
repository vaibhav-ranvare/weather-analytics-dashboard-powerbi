# weather-analytics-dashboard-powerbi
Interactive Weather Analytics Dashboard built using Power BI with current weather, hourly and daily forecasts, AQI analysis, and city-wise weather insights.

## Project Overview

WeatherVision is an interactive Power BI dashboard designed to visualize current weather conditions, hourly and daily forecasts, and environmental metrics. The dashboard enables users to monitor weather trends across multiple cities through dynamic visualizations and interactive filters.

---

## Features

-  Current Temperature
-  Weather Condition
-  Humidity
-  Wind Speed
-  Visibility
-  UV Index
-  Chance of Rain
-  Sunrise & Sunset
-  Air Quality Index (AQI)
-  Hourly Forecast
-  Daily Forecast
-  Multi-City Weather Comparison
-  Interactive Filters and Slicers

---

##  Technologies Used

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX

##  Dataset
